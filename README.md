**Employee Attrition Prediction using Machine Learning**

Project Overview:

Employee attrition is a major challenge for organizations. This project aims to build a machine learning model that predicts whether an employee is likely to leave the company based on various work-related factors.

Objectives:

Identify key factors influencing employee attrition
Analyze the impact of job satisfaction, salary, and work conditions
Build a predictive model to classify employees
Provide insights to improve employee retention

Dataset:

Dataset Name: HR Analytics Dataset
Total Records: 14,999
Features: 9 input features + 1 target variable

Technologies Used:

Python
Pandas
NumPy
Matplotlib & Seaborn
Scikit-learn

Project Workflow:

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Model Building
Model Evaluation

Machine Learning Models Used:
Logistic Regression
Decision Tree
Random Forest

Model Performance:

Best Model: Random Forest
Accuracy: 99%

Key Insights:

Employees with low satisfaction are more likely to leave
Long working hours increase attrition risk
Lack of promotion affects retention
Salary and job satisfaction are major factors

Conclusion:

This project helps organizations identify employees at risk of leaving and take proactive measures to improve retention and workplace satisfaction.
